Java TCP Server (Multi vs Single Threaded):

A simple Java project demonstrating socket programming and the difference between multi-threaded and single-threaded servers.

🧵 MultiThreadedWebServer
Handles multiple clients concurrently (one thread per client)
Runs on port 8010
Client simulates ~100 parallel requests


🔁 SingleThreadedWebServer
Processes one client at a time (sequential)
Runs on port 8010
Best for learning basics
